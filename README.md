# UCI Adult Income Dataset Exploratory Analysis

An analytical data science project exploring demographic profiles, occupational trends, investment traits, and economic disparities leveraging the classic **UCI Adult Income Census Dataset** using Python, Pandas, Matplotlib, and Seaborn.

## 🚀 Key Investigative Scopes
1. **Target Stratification:** Tracking structural class imbalance between income thresholds (`<=50K` vs `>50K`).
2. **Demographic Disparities:** Percent-normalized exploration across variables like Gender, Race, and Marriage.
3. **Age Trajectory Analysis:** Tracking peak earning periods through structural population distribution density curves.
4. **Labor Operations:** Evaluating occupational dependencies with hours-per-week variance metrics.
5. **Capital Tracking:** Pinpointing investment yield behaviors against life staging markers.
